# EarthRoots Philippines

EarthRoots Philippines is a modern frontend-only environmental website focused on tree planting and reforestation in the Philippines.

The platform features:
- Interactive 3D Earth globe
- Real Philippine tree planting locations
- Animated environmental visuals
- Parallax and cinematic effects
- Donation-focused advocacy sections
- Responsive modern UI

Built with:
- Next.js
- React
- Tailwind CSS
- Three.js
- React Three Fiber
- Framer Motion

Goal:
Help raise awareness and support reforestation efforts across the Philippines through an immersive web experience.
